# MERN Ecommerce Website

# Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js

# Features
- a responsive and clean UI using Tailwind CSS.
- users can add items to the cart and can place orders.
- Admin panel allows admin to add, remove items, update user’s order with notifications using React-Toastify.

# Folder Structure
frontend/ – React frontend  
backend/ – Express & Node backend  
admin/ – Admin panel
